# 智谱 GLM5.2自动助手 使用说明

智谱 GLM5.2 Coding Plan 自动抢购助手。提供两种安装方式，**推荐在 EasyBR 指纹浏览器中使用**。

---

## 方式一：在 EasyBR / eBrower 指纹浏览器中使用（推荐）

适合多账号、多窗口同时监控，自带环境隔离。  支持mac 和Windows。

### 安装

- 官网：[https://www.ebrower.com/](https://www.ebrower.com/)
- 浏览器使用文档：https://www.ebrower.com/2026/helperdoc/2-easybr-liu-lan-qi-shi-yong-liu-cheng/

1. 注册登录 EasyBR（[官网](https://www.ebrower.com/)），默认提供 **5 个免费环境**。可以多开
2. 打开「扩展中心」，确认本扩展可用。
3. 新建浏览器环境时勾选本扩展，系统自动安装。
4. 打开环境，扩展即已就绪。

### 多开使用

每个环境独立指纹、Cookie 和登录态，可分别监控不同套餐：
- 环境 A 监控 Max
- 环境 B 监控 Pro
- 环境 C 监控 Lite

### 步骤

1. 在每个环境登录 BigModel 账号，打开：
   ```text
   https://bigmodel.cn/glm-coding?plantype=personal
   ```
2. 点击扩展图标打开弹窗，选择目标套餐。
3. 开启总开关，开始监控。

## 方式二：Chrome / Edge 等浏览器手动安装

适合单浏览器使用。

### 下载扩展包

```text
https://ebrower.com/crx/glmhelp.zip
```

解压后得到 `glmhelp.crx` 和 `glmhelp/` 文件夹，任选其一。

### 安装（文件夹方式，推荐）

1. 打开 `chrome://extensions/`，开启右上角「开发者模式」。
2. 点击「加载已解压的扩展程序」，选择 `glmhelp/` 文件夹。

### 安装（crx 方式）

1. 打开 `chrome://extensions/`，开启「开发者模式」。
2. 将 `glmhelp.crx` 拖入页面，确认添加。
3. 如拖拽无反应，改用文件夹方式。

### 使用

1. 打开 `https://bigmodel.cn/glm-coding?plantype=personal` 并登录。
2. 点击扩展图标打开弹窗。
3. 选择目标套餐（可多选），按需开启自动点击 / 通知 / 声音。
4. 打开总开关，开始监控。

---

## 弹窗功能

| 功能 | 说明 |
|------|------|
| 总开关 | 开启/暂停监控 |
| 目标套餐 | 选择监控 Lite / Pro / Max，可多选 |
| 自动点击购买 | 可购买时自动点击（仍需手动支付） |
| 通知提醒 | 弹出系统通知 |
| 声音提示 | 播放提示音 |
| 日志 | 实时监控状态与触发记录 |

---

## 常见问题

- **图标看不到**：被收进扩展拼图按钮，点击固定即可。
- **拖拽 crx 无反应**：改用文件夹方式安装。
- **页面没有高亮**：确认已登录且在 `glm-coding` 页面，刷新重试。
- **EasyBR 环境扩展未生效**：确认新建环境时已勾选本扩展。
