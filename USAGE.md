# 智谱 GLM5.2自动助手 使用说明

智谱 GLM5.2 Coding Plan 自动抢购助手 Chrome 扩展的安装与使用指南。

提供两种安装方式：

- 方式一：以 Chrome 扩展形式手动安装（适合单浏览器使用）
- 方式二：在 EasyBR / eBrower 指纹浏览器扩展中心中使用（适合多账号、多窗口同时监控）

---

## 一、扩展形式安装（Chrome / Edge 等谷歌内核浏览器）

### 1. 下载扩展包

下载地址：

```text
https://ebrower.com/crx/glmhelp.zip
```

解压后包含两种格式，任选其一安装：

- `glmhelp.crx`：打包好的扩展安装包
- `glmhelp/` 文件夹：未打包的扩展目录

### 2A. 用文件夹安装（推荐，最稳定）

1. 打开扩展管理页：

   ```text
   chrome://extensions/
   ```

2. 打开右上角的「开发者模式」开关。
3. 点击「加载已解压的扩展程序」。
4. 选择解压出来的 `glmhelp/` 文件夹。
5. 加载完成后，工具栏会出现扩展图标。

### 2B. 用 crx 安装

1. 打开扩展管理页：

   ```text
   chrome://extensions/
   ```

2. 打开右上角的「开发者模式」开关。
3. 把 `glmhelp.crx` 文件直接拖拽到该页面。
4. 在弹出的确认框中点击「添加扩展程序」。

> 提示：部分新版本 Chrome 对非商店来源的 crx 有限制，如果拖拽 crx 没反应，请改用「2A. 用文件夹安装」。

### 3. 开始使用

1. 打开目标页面：

   ```text
   https://bigmodel.cn/glm-coding?plantype=personal
   ```

2. 登录你的 BigModel 账号。
3. 点击工具栏中的扩展图标，打开弹窗。
4. 在「目标套餐」中选择要监控的套餐（Lite / Pro / Max，可多选）。
5. 按需开启「自动点击购买」「通知提醒」「声音提示」。
6. 打开右上角总开关，扩展开始监控。
7. 被监控的套餐卡片会在页面上高亮显示，弹窗会展示实时状态和日志。

> 说明：扩展只在套餐由售罄变为可购买时提醒并辅助点击；是否完成支付仍由你手动确认。

## 二、在 EasyBR / eBrower 指纹浏览器中使用（推荐多开）

EasyBR / eBrower 自带扩展中心，可以在新建浏览器环境时直接选择扩展，由系统自动安装到该环境，无需手动加载。适合多账号、多窗口同时监控不同套餐。

- 官网：[https://www.ebrower.com/](https://www.ebrower.com/)
- 浏览器使用：https://www.ebrower.com/2026/helperdoc/2-easybr-liu-lan-qi-shi-yong-liu-cheng/https://www.ebrower.com/helperdoc/apidoc.html)

### 1. 注册并登录 EasyBR / eBrower

1. 打开官网 [https://www.ebrower.com/](https://www.ebrower.com/)，用邮箱注册一个账号。
2. 下载并安装 EasyBR / eBrower 客户端。
3. 用刚注册的账号登录客户端。

> 免费额度：注册后默认提供 **5 个浏览器环境**，免费使用，基本够用（足以分别监控 Lite / Pro / Max 等套餐）。如需更多环境再考虑升级。

### 2. 在扩展中心确认扩展

1. 打开 EasyBR / eBrower 客户端。
2. 进入「扩展中心」。
3. 找到本扩展（如未内置，可按扩展中心的导入方式添加）。

### 3. 新建环境时选择扩展（自动安装）

1. 在 EasyBR / eBrower 中「新建浏览器环境」。
2. 在环境配置的扩展选项中，勾选本扩展。
3. 创建环境后，系统会自动把扩展安装到该环境中，无需手动加载。
4. 打开该环境的浏览器，扩展即已就绪。

### 4. 多开使用

每个 EasyBR / eBrower 环境拥有独立指纹、独立 Cookie、独立登录态和独立扩展配置，因此可以：

- 用环境 A 监控 Max 套餐
- 用环境 B 监控 Pro 套餐
- 用环境 C 监控 Lite 套餐
- 多个账号分别登录 `bigmodel.cn`
- 多个窗口同时运行，互不干扰

推荐流程：

1. 注册并登录 EasyBR / eBrower（注册后默认有 5 个免费环境，基本够用）。
2. 在扩展中心确认本扩展可用。
3. 新建多个浏览器环境，每个环境勾选本扩展。
4. 在每个环境分别登录不同 BigModel 账号，打开：

   ```text
   https://bigmodel.cn/glm-coding?plantype=personal
   ```

5. 在每个环境的扩展弹窗中选择该环境要监控的目标套餐。
6. 保持窗口运行，即可实现多浏览器、多账号、多套餐同时监控。

> 注意：多开只是提供隔离的浏览器环境；是否开启自动点击、是否确认订单，仍应由你根据目标网站规则和自身需求谨慎决定。

## 三、弹窗功能说明

- **总开关**：开启 / 暂停监控。
- **目标套餐**：选择要监控的套餐，可多选；被选中的套餐会在页面上高亮。
- **自动点击购买**：套餐变为可购买时自动触发点击（仍需手动完成支付）。
- **通知提醒**：套餐可购买时弹出系统通知。
- **声音提示**：套餐可购买时播放提示音。
- **补货前 N 分钟高频检查**：临近补货时间时提高检查频率。
- **日志**：展示监控状态、套餐状态变化和触发记录。

## 四、常见问题

- **扩展加载后没有图标**：图标可能被收进工具栏的扩展拼图按钮中，点击拼图图标并固定本扩展即可。
- **拖拽 crx 没反应**：改用「2A. 用文件夹安装」加载 `glmhelp/` 文件夹。
- **页面没有高亮 / 没有读到套餐**：确认已登录 BigModel，且当前在 `glm-coding` 页面；刷新页面后重试。
- **EasyBR 环境里扩展没生效**：确认新建环境时已勾选本扩展，或在该环境的扩展管理中确认扩展已启用。
